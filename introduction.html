ゲームの編集作業をするまえに準備しておかなければならないことをまとめました。

Step 1 GitHubの導入


0．

GitHub は ファイルを複数人で管理するソフトウェアなの。みんなで同じファイルを編集するために、みんなのPCでGitHub やGitHub Desktop を使えるようにしてもらうよ。。

1．GitHubのアカウント作成（持っている人はスキップ）

GitHubを使うには、アカウントの登録が必要だから、GitHubのアカウントを持っていないお友達は、GitHubにアクセスして、GitHubのアカウントを作ってね。

1．1 Collaboratorになる

サークルのDiscordにGitHubのユーザー名を送ってね。そうすると、サークル長から、GitHubに登録したメールアドレスに、Collaboratorの招待が送られてくると思うよ。Collaboratorになるとレポジトリのファイルを編集できるようになるよ。

1．2 Unityのバージョンを確認する。

Unityのバージョンが合っていないと、共同で編集できなくなるよ。
Unity Hub を開いて、左のリストの下の方にある、「インストール」を選んで、「2020.3.15」が入っているかどうか確認してね。
入っていなかった場合、右上の「インストール」を選んで、「Unity 2020.3.15」を加えてね。「次へ」を押すと、モジュールを追加すうる画面になるので、下の方に「Web GL Build Support」があるので、それを選択して、「次へ」を押してね。
※注意：全ての作業が終わるまでUnityの編集画面は開かないでね。

2．GitHub Desktop のインストール

GitHub Desktop はGitHubをわかりやすく使うためのツールだよ。GitHub Desktopからインストールしてね。

3．GitHub でゲームのファイルにアクセスする。

サークルのゲームのレポジトリにアクセスしてね。平野くんが作ったレポジトリ（共有しているフォルダ）に移動するよ。このページの右の方にある、緑色の、「Code」というボタンを押して、「Open with GitHub Desktop」というボタンを押してね。そうすると、GitHub Desktopが開いて、「Clone a repository」というウインドウが出るよ。Repository URL はいじらないでね。下にある「Local path」に、ゲームのファイルを格納したい場所を入力してね。 「Clone」を押すと、サークルのゲームのフォルダが選択した場所にクローンされるよ。
クローンの説明→（○）

4．Branchを作成する。

Branchとは、端的に言えば、それぞれのファイルを編集する際に、更新するファイルが競合しないように、編集する部屋を分けるような事だよ。
GitHub Desktop を開いて、左上のボタンから、サークルのレポジトリに移動してね。隣のボタンは「Current branch」のはずなので、そこのボタンを押そう。ウインドウが出るはずなので、「New branch」を押そう。Create a branch というウインドウが出るので、「name」に自分の名前を入れたbranchの名前を設定してね。
次に、先程作成したbranchに移動してね。＊全ての作業は自分のbranchで行ってください。

Step 2 Unityで編集をできるようにする。


1．ゲームのファイルをUnity Hubに追加する。

Unity Hub を開くと、「プロジェクト」という画面が表示されるので、右上の「リストに追加」を押してね。エクスプローラーが表示されるので、先程作成したクローンのフォルダを選択してね。そうすると、「プロジェクト」にサークルのゲームのフォルダが追加されるよ。

☆ ファイルを編集・更新する

＊ファイルを編集する前に、GitHub Desktopで、今いるブランチが自分のものである事を必ず確認してね。自分のブランチ以外の場所で編集作業をしないでください！

1．Unity での編集を反映させる前に、Unity を閉じておいてね。
2．GitHub Desktopの「Changes」に、編集したファイルが表示されるよ。
＊もし、ファイル名の末尾に「.log」があるファイルがある場合、チェックを外しましょう。Discardしても問題ありません。
3．右上の「Fetch Origin」を押して、更新してね。 4．下の方にある「Commit to 〇〇」という青いボタンを押してね。そうすると、上の方に、「Push origin」というボタンが現れるので、このボタンを押すと、自分のブランチに変更内容が反映されるよ。

＊ゲームの編集そのものについての質問は平野さんにお願いします。
・誰がどのステージ（Scene）を編集するべきかについては、Discordに載っています。
