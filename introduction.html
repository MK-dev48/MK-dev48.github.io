Step 1 GitHubの導入 0． GitHub は ファイルを複数人で管理するソフトウェアなの。みんなで同じファイルを編集するために、みんなのPCにGitHubを入れてもらうよ。 1．GitHubのアカウント作成（持っている人はスキップ） GitHubを使うには、アカウントの登録が必要だから、GitHubのアカウントを持っていないお友達は、（https://github.com/）にアクセスして、GitHubのアカウントを作ってね。 1．1 Collaboratorになる サークルのDiscordにGitHubのユーザー名を送ってね。そうすると、平野くんから、GitHubに登録したメールアドレスに、Collaboratorの招待が送られてくると思うよ。Collaboratorになるとレポジトリのファイルを編集できるようになるよ。 1．2 Unityのバージョンを確認する。 Unityのバージョンが合っていないと、共同で編集できなくなるよ。 Unity Hub を開いて、左のリストの下の方にある、「インストール」を選んで、「2020.3.15f1」が入っているかどうか確認してね。 入っていなかった場合、右上の「インストール」を選んで、「Unity 2020.3.15f2」を加えてね。「次へ」を押すと、モジュールを追加すうる画面になるので、下の方に「Web GL Build Support」があるので、それを選択して、「次へ」を押してね。 ※注意：全ての作業が終わるまでUnityの編集画面は開かないでね。 2．GitHub Desktop のインストール GitHub Desktop はGitHubをわかりやすく使うためのツールだよ。（https://desktop.github.com/）からインストールしてね。 3．GitHub でゲームのファイルにアクセスする。 （https://github.com/Hirano1182/2DGame ）にアクセスしてね。平野くんが作ったレポジトリ（共有しているフォルダ）に移動するよ。このページの右の方にある、緑色の、「Code」というボタンを押して、「Open with GitHub Desktop」というボタンを押してね。そうすると、GitHub Desktopが開いて、「Clone a repository」というウインドウが出るよ。Repository URL はいじらないでね。下にある「Local path」に、ゲームのファイルを格納したい場所を入力してね。 「Clone」を押すと、サークルのゲームのフォルダが選択した場所にクローンされるよ。 クローンの説明→（https://docs.github.com/ja/github/creating-cloning-and-archiving-repositories/cloning-a-repository-from-github/cloning-a-repository ） 4．ゲームのファイルをUnity Hubに追加する。 Unity Hub を開くと、「プロジェクト」という画面が追加されるので、右上の「リストに追加」を押してね。エクスプローラーが表示されるので、先程作成したクローンのフォルダを選択してね。そうすると、「プロジェクト」にサークルのゲームのフォルダが追加されるよ。 5．Branchを作成する。 Branchとは、端的に言えば、それぞれのファイルを編集する際に、更新するファイルが競合しないように、編集する部屋を分けるような事だよ。 GitHub Desktop を開いて、左上のボタンから、サークルのレポジトリに移動してね。隣のボタンは「Current branch」のはずなので、そこのボタンを押そう。ウインドウが出るはずなので、「New branch」を押そう。Create a branch というウインドウが出るので、「name」に自分の名前を入れたbranchの名前を設定してね。 次に、先程作成したbranchに移動してね。＊全ての作業は自分のbranchで行ってください。
